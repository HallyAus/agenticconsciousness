Tools Page: Which layout approach?

The current page uses a grid of cards that expand into a panel below. Here are 3 alternative directions.

A

Interactive Showcase (Hero per Tool)

Each tool gets a full-width hero section as you scroll down the page. Live demo embedded inline — no click-to-expand. User sees the tool working immediately with example data. Think "product landing page" where each section sells one tool.

Pros

  • Maximum visual impact — each tool feels important
  • Live demos visible without clicking anything
  • Great for SEO (all content server-rendered)
  • Scroll-based discovery feels natural

Cons

  • Very long page (8 sections)
  • Hard to compare tools quickly
  • Heavy page weight (all tools loaded)
B

Tool Cards with Preview + Instant Try

Larger cards in a 2-column grid. Each card shows: tool name, what it does, a mini visual preview of sample output, and a prominent "Try it free" button. Clicking opens a full-screen modal/overlay with the tool interface. Cards sell the outcome, not the input.

Pros

  • Users see WHAT they'll get before committing
  • Output previews create desire ("I want that")
  • 2-col grid is scannable but not overwhelming
  • Modal keeps context (user doesn't lose their place)

Cons

  • Preview screenshots need to be compelling
  • Modal can feel jarring on mobile
  • More design work per card
C

Problem-First Flow (Task Selector)

Instead of listing tools, ask the user "What do you need to do?" with big clickable task blocks: "Scan an invoice", "Write an email", "Analyse a competitor", etc. Each task block shows the outcome and time saved. Clicking goes straight into the tool. The page sells solutions, not software.

Pros

  • User-centric — speaks their language
  • Reduces decision fatigue (task → tool mapping)
  • Strongest conversion framing ("solve your problem")
  • Easy to A/B test different task descriptions

Cons

  • Some tools serve multiple tasks (harder to categorise)
  • Less "impressive" than showing all 8 tools at once
  • May feel too simple for power users
